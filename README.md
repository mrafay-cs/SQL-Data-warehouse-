Sql-data-warehouse-ERP-CRM-project

An end-to-end SQL Server data warehouse project that uses Medallion Architecture to organise ERP and CRM data into Bronze, Silver, and Gold layers. The project focuses on building a clean and structured data pipeline for sales analytics and business reporting.

The solution includes importing CSV data, cleansing and transforming raw data, integrating ERP and CRM sources, and creating analytics-ready data models. It demonstrates key data engineering concepts such as ETL, data quality handling, dimensional modelling, and structured warehouse design.

The analytics layer supports business insights across customer behaviour, product performance, and sales trends, helping stakeholders monitor key metrics and make data-driven decisions.
